Прелоадер:

1. Створити в html div з класом.
2. Застосувати до діва css:
    2.1 Зробити його зафіксованим
    (position:fixed; top:0; left:0; )
    2.2 Задаємо ширину 100% і висоту 100vh
    2.3 Заливаємо кольором (бо він прозорий)
    2.4 вмикаємо флексбокси і ставимо по центру
    2.5 Потрібно зафіксувати зверху (z-index:100500)
3. Шукаємо в інтернеті "прелоадер css"  і копіюємо його код
4. Джава скрипт
    4.1 Створити змінну для завнішнього перлоадера 
    4.2 Створити функцію на подію "сторінка поргрузилася"
    window.addEventListener('load', function ({  ...   }))
    4.3 Виключення перлоадера зробимо через 1 секунду, щоб 
    його точно будло видно 
        setInterval(() => {
            ....
        }, 1000);
    4.4 Вимикаємо видимість всього прелоадеру
        змінна.style.display = 'none'
